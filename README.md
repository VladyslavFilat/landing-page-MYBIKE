
# 🔳 MYBIKE  Landing Page

This project is a modern, responsive landing page created to showcase products from the "MyBike" store. It is designed in a clean, minimalist style that aligns with the store’s identity, with a focus on user experience and visual appeal.

## 🚀 Getting Started
  To get a local copy up and running, follow these simple steps.


  ### Clone the repository:

    git clone https://github.com/VladyslavFilat/landing-page-MYBIKE.git

  ### Open the project folder:

    cd landing-page-MYBIKE

  ### Install NPM packages:

    npm install

   ### Start the project:

    npm start

## 💻 Technologies Used

  - HTML5 — for semantic structure and content of the page
  - SCSS (Sass) — for writing clean, maintainable, and scalable styles
  - CSS3 — for layout, including Flexbox and Grid, and responsive design
  - Git — for version control and tracking changes
  - GitHub Pages — for deploying and hosting the live project
